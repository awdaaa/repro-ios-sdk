## Usage

To run the example project; clone the repo, and run `pod install` from the Example directory first.

## Required frameworks

* AVFoundation.framework
* CoreGraphics.framework
* CoreMedia.framework
* CoreVideo.framework
* Repro.framework
* UIKit.framework
